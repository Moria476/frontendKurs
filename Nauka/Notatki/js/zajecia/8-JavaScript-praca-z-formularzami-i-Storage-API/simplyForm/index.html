Polecenie:

Twoim zadaniem jest zbudować formularz.

Formularz powinien mieć następujące pola: Imie, Pseudonim, Data urodzenia (wykorzystaj type='date'), Pytanie czy akceptuje warunki rejestracji (checkbox).

Jeżeli użytkownik akceptuje warunki, wyświetl mu dodatkową klauzule informującą o tym, że może w każdej chwili zrezygnować z umowy, jezeli nie zaznaczyl checkboxa - ukryj dodatkowa klauzule

Po kliknięciu submit wyświetl wszystkie zebrane informacje w konsoli w formacie JSON, gdzie klucz to nazwa pola.
